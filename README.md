# Node Mongo Express API

Node api to fetch records from DB with test cases using JEST and code coverage

## Start
- create a `deploy.sh` file
- add following envs

```
  export MONGO_URI=
  export APP_PORT=
  npm run start
```

- run the api with `sh deploy.sh`

## Test

- using below command tp run test cases

```
npm run test
```
